新規登録の流れ

{% comment %}

ゲームプレイに必要なもの

ゲームをプレイするために、次のものを用意しましょう！

  1. メールアドレス（個人、PCで利用できるメールアドレスを推奨）
  2. インターネット接続環境
  3. インターネットブラウザを搭載した機器（※注１）

※注１：動作確認は「このサイトについて」に記載しています。

FAQ? 月額利用料金はかかるの？
このゲームをプレイする料金は一切かかりません。

{% endcomment %}

登録手順

Grand Blazeへの新規キャラクターの登録は、以下の手順で行います。

アカウント情報、キャラクター設定の入力
Grand Blazeでプレイを行うあなたのキャラクター情報と、あなたの情報（メールアドレスやログインパスワードなど）を登録します。すでに登録されているメールアドレスは使用できません。
利用できるメールアドレスに制限はありませんが、PC画面上で確認できるように最適化されているため、PCで使用できるメールアドレスを推奨します。
※携帯電話のメールアドレスでは、送信されてくるメールの内容が途切れてしまうこともあります。
すべての情報を入力したら、Grand Blaze会員規約の内容を確認し、「同意して登録する」をクリックします。
仮登録完了
仮登録が完了します。この段階ではまだ登録が完了していません。
登録したメールアドレス宛に「仮登録のお知らせ」が送信されますので、メール本文中のアドレスを開き、本登録を完了すると、キャラクターが完成します。
本登録完了
キャラクターの登録申請が行われました。
無事に登録が行われていると、登録メールアドレス宛てに「新規登録完了のお知らせ」が送られてきます。
このお知らせが送られてきたら、まずは最初の「継続登録」を行いましょう。
はじめての継続登録
これからゲームを続けてプレイする上で基本フローとなる「継続登録」を行います。
継続登録では、主に次の冒険で向かう「移動先」を決定します。
はじめての継続登録では、移動先はクエスト「ようこそギルドへ！」のマーク「はじまりの場所」に固定されています。
まずは、この「はじまりの場所」へ向かい、Grand Blazeでの冒険をはじめることとしましょう！
※このタイプのゲームに慣れている場合、ここでパーティを編成することも可能です。詳細はパーティの編成をご覧ください。
※この継続登録を行わなくても、冒険は「はじまりの場所」からはじまります。
ログインをしよう！
継続登録を行うために、まずログインを行いましょう。
ログインするには、画面右上にある「ログイン」をクリックし、登録したメールアドレスとパスワードでログインします。
ログインすることでサイト内の様々な機能にアクセスできるようになります！
継続登録完了
継続登録を登録したら、あとは次の更新日まで待つだけです。
冒険結果の結果発表後に、そのエントリー番号で登録されていることが確認できれば、Grand Blazeの世界へ自由に羽ばたくことが可能です。
おつかれさまでした！
